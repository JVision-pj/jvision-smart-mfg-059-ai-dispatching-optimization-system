# AI派工優化洞察站

以AI演算法動態優化現場派工與排程決策，取代傳統固定規則派工邏輯之系統。

- 原始系統：AI派工優化系統（AI Dispatching Optimization System）
- 產業分類：智慧製造｜製造管理
- 使用部門：生管、製造工程、現場班組長
- 工作流程：即時現況數據輸入 → AI派工模型運算 → 派工建議產出 → 現場執行 → 效益回饋優化模型
- AI 應用：強化學習動態優化派工策略、AI持續學習現場執行結果改善模型準確度
- 主要 KPI：派工優化達成率、平均等待時間縮短率、交期達成率
- 代表產品：多為APS/MES廠商新增AI功能模組，此類獨立AI派工優化產品市場尚在發展初期，無成熟獨立知名品牌
